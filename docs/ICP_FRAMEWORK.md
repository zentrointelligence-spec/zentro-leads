# ICP Framework

_Source: `Downloads/Zentro_Leads_Master_Knowledge_Base.docx`._

B2B and B2C ideal customer profile frameworks used by the Zentro Leads ICP builder.

## Section 2 — The 10-Stage Lead Generation Framework

This framework covers every stage from identifying who our customers' ideal clients are, all the way through to measuring results. Each stage maps directly to a feature or component inside Zentro Leads.

### Stage 1 — ICP (Ideal Customer Profile)

The ICP defines exactly who the insurance agency is trying to reach. Without a sharp ICP, lead generation is just guessing. Our ICP Builder feature automates this step.

#### B2B ICP Framework — Corporate Insurance Leads

| ICP Dimension | Insurance-Specific Application |
| --- | --- |
| Industry | Manufacturing, construction, F&B, retail, logistics, healthcare |
| Company size | 10–200 employees (group medical trigger point) |
| Revenue range | RM 1M–50M / ₹1Cr–50Cr |
| Decision maker | CEO, HR Manager, Finance Director, Company Secretary |
| Pain point | Staff medical coverage, business continuity, asset protection |
| Buying trigger | Company growth, new hire surge, regulatory compliance |
| Location | Selangor, KL, Penang (MY) \| Mumbai, Delhi, Bangalore (IN) |

#### B2C ICP Framework — Individual Insurance Leads

| ICP Dimension | Insurance-Specific Application |
| --- | --- |
| Life stage | 22–55 years old, working adults |
| Income bracket | RM 3,000–15,000/month \| ₹25,000–150,000/month |
| Trigger event | Car purchase, home purchase, marriage, baby, job change |
| Insurance gap | Uninsured, underinsured, policy lapse |
| Geography | Urban + suburban — Klang Valley, Johor, Chennai, Hyderabad |
| Behaviour signal | New vehicle registered, property transaction, social life event |
| Ideal product | Motor, medical, life, home, personal accident |

#### How This Powers Our ICP Builder Feature

Our AI ICP Builder takes a one-line description from the insurance agent and generates a complete structured ICP automatically using Claude Sonnet. The agent types: 'I sell group medical insurance to SMEs in Kuala Lumpur' and receives a fully structured ICP with all dimensions filled, which then drives the lead generation query.
